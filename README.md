# stock-app

## Installation
- Install dependencies. Required after every clone.

```
npm install
```

- Compile and hot-reloads for development

```
npm run serve
```

## Build for Production

- Lint and fix files

```
npm run lint
```

- Compile and minify for production

```
npm run build
```
